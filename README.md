# UntitledUtils

A C++ library for my projects.

## UntitledLogger

Generic wrapper logger.

```cpp
#include <untitled/logger.hh>
using Log = Untitled:Console::Logger;

int main(void)
{
    Log::print("Hello, world!");
    return 0;
}
```

Features:
- Cards (aka tags)
- Colors
- Template
